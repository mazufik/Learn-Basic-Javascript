# Operator Matematika

Javascript mendukung banyak sekali operator matematika untuk tipe data number,
seperti :

- Operator Aritmatika
- Operator Augmented Assignments
- Operator Unary
- dan lain-lain

### Opeator Aritmatika

| Opeator | Keterangan  |
| :-----: | ----------- |
|    +    | Pertambahan |
|    -    | Pengurangan |
|   \*    | Perkalian   |
|  \*\*   | Exponensial |
|    /    | Pembagian   |
|    %    | Sisa Bagi   |
