# Operator Logika

- Operator logika adalah operator untuk dua buah data boolean
- Hasil dari operator logika adalah boolean lagi

| Operator | Keterangan |
| -------- | ---------- |
| &&       | Dan        |
|          | Atau       |
| !        | Kebalikan  |

### Operator && (Dan)

| Nilai 1 | Operator | Nilai 2 | Hasil |
| ------- | :------: | ------- | ----- |
| true    |    &&    | true    | true  |
| true    |    &&    | false   | false |
| false   |    &&    | true    | false |
| false   |    &&    | false   | false |

### Operator || (Atau)

| Nilai 1 | Operator | Nilai 2 | Hasil |
| ------- | :------: | ------- | ----- |
| true    |    OR    | true    | true  |
| true    |    OR    | false   | true  |
| false   |    OR    | true    | true  |
| false   |    OR    | false   | false |

### Operator Unary !

| Operator | Nilai | Hasil |
| :------: | ----- | ----- |
|    !     | true  | false |
|    !     | false | true  |
